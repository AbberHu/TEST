DXS 應援交換 · 安裝說明
===============================================

檔案內容：
  index.html              主程式
  manifest.json           PWA 設定檔
  sw.js                   離線快取 Service Worker
  doa.png / boo.png       應援吉祥物圖示
  icon-*.png              各種尺寸的 App 圖示

===============================================
安裝方式（3 個步驟）
===============================================

【步驟 1】把整個資料夾上傳到免費靜態網站

推薦 Netlify Drop（最簡單，不用註冊）：

  1. 電腦打開 https://app.netlify.com/drop
  2. 把整個資料夾（含全部檔案）拖進網頁
  3. 立刻會給你一個網址，例如：
     https://silly-name-abc123.netlify.app
  4. 記下這個網址

其他選項：
  - Cloudflare Pages（需註冊，但更快更穩）
  - Vercel、GitHub Pages 都可以

【步驟 2】手機打開網址

  用手機瀏覽器打開步驟 1 的網址：
  - iPhone → Safari
  - Android → Chrome

【步驟 3】加入主畫面

  iPhone (Safari)：
    網址列下方分享按鈕 → 「加入主畫面」→ 完成
    
  Android (Chrome)：
    右上角三點選單 → 「新增至主畫面」→ 完成
    （有時 Chrome 會自動跳出安裝提示）

完成後主畫面會有 D×S 星星圖示，
點下去像獨立 App 一樣打開，資料存在手機本地。

===============================================
資料備份
===============================================

App 內右上角「匯出 CSV」按鈕可隨時匯出全部記錄成
試算表檔，Google Sheets、Excel 都能開啟。

===============================================
換手機/清除瀏覽器資料時
===============================================

資料存在瀏覽器的 localStorage，清除瀏覽器資料
會一起清掉。建議定期匯出 CSV 備份。
